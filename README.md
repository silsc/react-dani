# Dani Salat 🎸

Portfolio page built with Create React App.

## Stack

👩🏻‍🎨 Designed in Figma

👷🏻‍♀️ Built with React using React Router DOM

🎨 Styled with [Sass](https://sass-lang.com/)

👩🏻‍💻 Deployed with Netlify.


## Project setup
```
npm install
```
```
npm start
```
